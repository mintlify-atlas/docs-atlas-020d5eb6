## Overview

The Essential Mod source code is made available under a Source Available License Agreement. This license demonstrates Essential's commitment to transparency while protecting their intellectual property rights.

The source code is provided primarily to allow users to:
- Audit the code for security and authenticity
- Verify that distributed binaries match the official source code
- Build the mod from source to confirm its authenticity

<Warning>
  This is a source-available license, not an open-source license. You cannot use, modify, or distribute the code.
</Warning>

## What you can do

Under the Essential Mod Source Available License Agreement, you are permitted to:

- **Audit the source code** - Review the code for security, quality, or educational purposes
- **Compile the source code** - Build the mod from source to confirm the authenticity of official releases
- **Create unmodified forks** - Fork the repository without making modifications (as of January 22, 2025)
- **Verify distributed binaries** - Audit compiled binaries to ensure they match the official source code
