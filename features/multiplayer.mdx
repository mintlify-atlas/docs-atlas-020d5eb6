## Getting started

<Tabs>
  <Tab title="Quick start">
    ### Starting a hosted session
    
    <Steps>
      <Step title="Open world or main menu">
        You can host from the main menu or while playing
      </Step>
      
      <Step title="Click 'Host World' button">
        - **Main menu**: "Host World" button for quick access

      </Step>
      
      <Step title="Invite friends">
        Select friends to invite from your friends list
      </Step>
    </Steps>
    
    <Info>
      All hosted sessions are invite-only for privacy and security. The "friends" privacy setting has been removed.
    </Info>
  </Tab>
  
  <Tab title="Main menu hosting">
    ### Host from main menu
    
    The main menu provides a dedicated "Host World" button for quick access:
    
    **Features:**
    - Quick world selection
    - Pre-configure settings before loading
    - Invite friends before world loads
    - View world information
    
    **Benefits:**
    - Faster hosting setup
    - No need to load world first
    - Configure everything in advance
    - Streamlined friend invites
  </Tab>
  
  <Tab title="In-game hosting">
    ### Host while playing
    
    You can also start hosting while already in a world:
    
    1. Open pause menu
    2. Select "Invite Friends" or "Host World" option
    3. Choose friends to invite
    4. Friends can join immediately
    
    **Use cases:**
    - Already playing and want to invite friends
    - Change hosting settings mid-session
    - Invite additional friends later

### Host commands

Hosts have access to special commands:

```bash
# Invite a player to your world
/essential invite <player>

# Kick a player from your world
/essential kick <player>

# Grant operator status to a player
/essential op <player>

# Remove operator status from a player
/essential deop <player>
```

<Info>
  Commands are available when cheats are enabled for the hosted world.
</Info>

## Connection system

### Connection quality

Essential uses advanced networking for optimal connections:

**Connection methods:**
1. **Direct connection** - Best performance, lowest latency
2. **Regional proxies** - Used when direct connection isn't possible
3. **Fallback routing** - Ensures connectivity in all scenarios

**Improvements:**
- Reduced latency for proxy connections
- Better connection quality worldwide
- Fewer timeout issues
- Improved stability

<Info>
  Players are no longer kicked from World Hosting when the host temporarily loses connection to Essential infrastructure.
</Info>

### Joining hosted worlds

**Players can join via:**
- Direct invite notifications
- Social Menu world invites
- Discord invites (with integration)
- Friends list (see active hosted worlds)

**Join process:**
1. Receive invite notification
2. Click "Join" button
3. Automatic connection handling
4. Load into host's world

## Resource pack sharing

### How it works

<Tabs>
  <Tab title="Enabling">
    ### Share your resource pack
    
    When hosting a world:
    
    1. Enable "Share Resource Pack" in world settings
    2. Essential packages your active resource pack
    3. Joining players automatically download it
    4. Pack applies when they join your world
    
    <Info>
      Resource pack sharing now supports outdated resource packs, ensuring compatibility.
    </Info>
  </Tab>
  
  <Tab title="Features">
    ### Resource pack features
    
    - **Automatic transfer** - No manual download needed
    - **Outdated pack support** - Works with older packs
    - **Compression** - Optimized transfer size
    - **No zipping when disabled** - Saves resources
    - **Seamless application** - Auto-applies on join
    
    <Note>
      Host resource packs are no longer unnecessarily zipped when "Share resource pack" is disabled.
    </Note>
  </Tab>
  
  <Tab title="Troubleshooting">
    ### Common issues
    
    <Accordion title="Game stuck after hosting with resource pack">
      **Problem:** Game freezes when opening singleplayer world after hosting with shared resource pack
      
      **Solution:** This has been fixed in recent Essential versions. Update to the latest version.
    </Accordion>
    
    <Accordion title="Resource pack not transferring">
      **Solutions:**
      - Verify "Share Resource Pack" is enabled
      - Check resource pack size (very large packs may timeout)
      - Ensure both players have stable internet
      - Try disabling and re-enabling sharing
    </Accordion>
  </Tab>
</Tabs>

## Troubleshooting

<Accordion title="Players timing out">
  **Solutions:**
  - This issue has been largely resolved with connection improvements
  - Ensure all players have stable internet
  - Try hosting from a different network if issues persist
  - Check firewall settings aren't blocking connections
</Accordion>

<Accordion title="Connection quality issues">
  **Improvements made:**
  - Regional proxies now provide better quality
  - Reduced latency in all connection modes
  - Better fallback routing
  
  **If still experiencing issues:**
  - Check network stability
  - Close bandwidth-heavy applications
  - Try direct connection if possible
</Accordion>

<Accordion title="Voice chat not working">
  **Plasmo Voice:**
  - Update to Essential latest version (2.x support fixed)
  - Verify Plasmo Voice is installed correctly
  - Check voice chat keybinds
  
  **Simple Voice Chat:**
  - Ensure mod is properly installed
  - Verify audio permissions
  - Check voice settings in-game
</Accordion>

<Accordion title="Can't create new world from host modal">
  **Feature:**
  A "Create New World" button has been added to the host world modal for convenience.
  
  If not visible, ensure you're running the latest Essential version.
</Accordion>
