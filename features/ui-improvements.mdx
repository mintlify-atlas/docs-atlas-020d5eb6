## Troubleshooting

<Accordion title="Buttons not showing correctly">
  **Solution:** Ensure "Use Minecraft button texture" setting matches your preference and resource pack is compatible.
</Accordion>

<Accordion title="Social Menu performance">
  **Solution:** Performance issues with long messages have been fixed. Update to the latest Essential version.
</Accordion>

<Accordion title="Keybinds not working">
  **Solutions:**
  - Check for keybind conflicts in Controls settings
  - Verify keybinds aren't disabled
  - Restart Minecraft after changing keybinds
</Accordion>

<Accordion title="Account switcher issues">
  **Solutions:**
  - Use "Refresh Session" button if session expired
  - Re-add account if authentication fails
  - Check Microsoft account status
</Accordion>
