<Tabs>

### Cosmetic customization

<AccordionGroup>
  <Accordion title="Color options" icon="palette">
    Customize cosmetic colors to match your style:
    
    - **Color picker** - Full RGB color selection
    - **Preset colors** - Quick color options
    - **Per-cosmetic** - Different colors for each item
    - **Live preview** - See changes in real-time
    
    <Note>
      Changing colors does not reset cosmetic animations - your items will continue animating smoothly.
    </Note>
  </Accordion>
  
  <Accordion title="Position adjustments" icon="arrows-up-down-left-right">
    Fine-tune cosmetic placement:
    
    - **Position offset** - Move cosmetics up/down/left/right
    - **Rotation** - Adjust angle and orientation
    - **Scale** - Resize cosmetic items
    - **Reset** - Return to default positioning
  </Accordion>
  
  <Accordion title="Selection & equipping" icon="hand-pointer">
    Interact with cosmetics easily:
    
    - **Click to select** - Select cosmetics on character preview
    - **Quick equip** - Click in grid to equip/unequip
    - **Multi-slot** - Wear multiple compatible cosmetics
    - **Hover outline** - Visual selection indicator
    
    <Note>
      Cosmetic hover outlines work correctly on all Minecraft versions including macOS on 1.8.9-1.16.5.
    </Note>
  </Accordion>
  
  <Accordion title="Rarity tiers" icon="star">
    Cosmetics are organized by rarity:
    
    - **Common** - Basic cosmetics
    - **Uncommon** - Enhanced items
    - **Rare** - Special cosmetics
    - **Epic** - Premium items
    - **Legendary** - Exclusive cosmetics
    
    Rarity affects appearance, animations, and availability.
  </Accordion>
</AccordionGroup>

## Emotes system

### Using emotes

<Tabs>
  <Tab title="Emote wheel">
    ### In-game emote wheel
    
    Access emotes during gameplay:
    
    - **Open wheel** - Hold emote wheel keybind
    - **Select emote** - Move mouse to highlight and release
    - **Play emote** - Character performs selected emote
    - **Cancel** - Move to center and release
    
    **Emote wheel features:**
    - Switch between wheels in-game
    - Scroll to cycle through wheels
    - Customizable slot assignments
    - Quick favorite access
  </Tab>
  
  <Tab title="Configuration">
    ### Emote wheel setup
    
    Customize your emote wheels:
    
    - **Multiple wheels** - Create multiple wheel configurations
    - **Slot assignment** - Drag emotes to wheel slots
    - **Wheel switching** - Change active wheel in wardrobe or in-game
    - **Favorites** - Quick access to favorite emotes
    
    <Note>
      A warning appears in the wardrobe if your emote wheel keybind conflicts with another keybind.
    </Note>
  </Tab>
  
  <Tab title="Viewing options">
    ### Emote camera
    
    Control how you view emotes:
    
    - **Third person** - View from behind character
    - **First person preview** - Optional first-person view
    - **Auto-camera** - Automatic camera positioning
    - **Manual control** - Free camera movement
    
    **Settings:**
    - "Play emotes in third person view" option
    - First-person preview toggle
    - Camera behavior customization
  </Tab>
</Tabs>

### Emote features

- **Rarity tiers** - Organized by rarity like cosmetics
- **Bundle previews** - Emotes play in bundle previews (looping fixed)
- **Gifting** - Send emotes to friends
- **Animations** - Full character animations
- **Compatibility** - Works with cosmetics equipped

<Info>
  Parrots now animate correctly during emotes on 1.16+ Forge with OptiFine installed.
</Info>

## Skins & capes

### Skin library

<Tabs>
  <Tab title="Managing skins">
    ### Skin management
    
    The skin library provides comprehensive skin management:
    
    **Add skins via:**
    - Upload file (PNG)
    - Paste link (URL)
    - Username lookup
    - Direct file selection
    
    **Organize skins:**

    - Name and categorize
    - Quick preview
    - Delete unwanted skins
  </Tab>
  
  <Tab title="Skin features">
    ### Advanced features
    
    - **Outfit locking** - Lock skins to specific outfits
    - **Mojang sync** - Sync with Mojang skin servers
    - **Sharing** - Share via Social Menu or link
    - **Usage indicator** - See which outfits use a skin
    - **Favorites** - Mark frequently used skins
    
    <Warning>
      On Minecraft 1.20.2 to 1.21.8, skin textures may be removed after opening the wardrobe. This has been fixed in recent updates.
    </Warning>
  </Tab>
  
  <Tab title="Uploading">
    ### Uploading skins
    
    Upload custom skins to Mojang:
    
    - Select skin from library
    - Click upload to Mojang
    - Automatic sync
    - Processing notification
    
    <Note>
      The game freezing when uploading skins has been fixed in recent Essential versions.
    </Note>
  </Tab>
</Tabs>

## Outfits system

### Creating outfits

<Steps>
  <Step title="Select items">
    Equip cosmetics, skin, and cape combinations you want to save
  </Step>
  
  <Step title="Create outfit">

  <Step title="Name outfit">

  </Step>

## Compatibility

### Mod compatibility

<Check>**Compatible with:**</Check>
- OptiFine (emote animations fixed on 1.16+ Forge)
- Armor mods (configurable interaction)
- Skin mods (managed through settings)
- Most visual enhancement mods

<Warning>**Known conflicts:**</Warning>

- Entity Model Features with Quark (Q key crash fixed)

### Anti-cheat compatibility

<Info>
  Essential no longer sends client settings packets on all cosmetic updates, avoiding false triggers with some Anti-Cheat systems.
</Info>
